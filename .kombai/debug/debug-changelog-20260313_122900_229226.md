- **CORS issue:** `ssd-api.jpl.nasa.gov` has no `Access-Control-Allow-Origin` header → browser blocked. Fixed via Vite proxy for dev + Netlify `_redirects` for prod.
- **Turin Scale NaN:** Some Sentry objects have missing `ts_max` field → fallback `|| 0` added.
- **Name truncation:** Original `flex items-start justify-between` layout caused name to be truncated to "2..." with `truncate`. Fixed by using `flex-col` layout so badge sits below the name.
- **Cache key:** Bumped to `sentry_v2` to bust old cache that held the direct (CORS-blocked) URL.
