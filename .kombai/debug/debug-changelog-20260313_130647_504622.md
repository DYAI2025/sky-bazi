## Changes Made

### 1. Added infrastructure to `src/services/nasa.ts`
- **File:** `src/services/nasa.ts`
- **Change (In-flight deduplication):** Added module-level `inFlight: Map<string, Promise<any>>` and `fetchOnce(key, fn)` wrapper. If multiple React components mount simultaneously and call the same fetch function, they now share a single in-flight Promise instead of firing N identical requests.
- **Change (429 backoff):** Added `isRateLimited(cacheKey)` / `markRateLimited(cacheKey)` helpers using a `rl:` prefixed localStorage key with 30-min TTL. When any endpoint returns HTTP 429, its key is marked and all subsequent calls skip the fetch for 30 minutes.
- **Change (IS_DEMO_KEY export):** Added `export const IS_DEMO_KEY = API_KEY === "DEMO_KEY"` for UI hint.
- **Revert:** Remove `IS_DEMO_KEY`, `RATE_LIMIT_TTL`, `isRateLimited`, `markRateLimited`, `inFlight`, `fetchOnce`.
